Prompt Architecture

The modular nested prompt system

Every model call is one system prompt, assembled at runtime from a fixed base, a set of nested fragments that switch on only when an option calls for them, and a shared block of quality constraints — then locked to an exact JSON contract. Each tab is a view of that system; click any box to unfold it. The wiring re-routes as the diagram expands.
